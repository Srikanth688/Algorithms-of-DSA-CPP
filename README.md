# 🚀 Data Structures & Algorithms (DSA) using C++

## 📌 Overview

This repository contains implementations of **Data Structures and Algorithms (DSA)** and **Design and Analysis of Algorithms (DAA)** concepts using **C++**.
It is designed for **learning, practice, and interview preparation**.

---

## 🎯 Objectives

* Understand core DSA concepts
* Practice problem-solving using C++
* Prepare for coding interviews and competitive exams
* Improve logical thinking and algorithm design skills

---

## 🧠 Topics Covered

### 🔹 Data Structures

* Arrays
* Strings
* Linked List (Singly, Doubly)
* Stack
* Queue
* Trees (Binary Tree, BST)
* Heap (Min Heap, Max Heap)
* Graphs
* Hashing

| Algorithm     | Time Complexity |

| Binary Search | O(log n)        |
